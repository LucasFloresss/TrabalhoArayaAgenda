Instrução internas (num é pro senhor nao professor):

- mysql aberto e conectado com XAMPP (tem que criar a db lá também)

- vai no navegador e digita http://localhost:3000/ e bufo
- lembra de ativar o auto-save pra qualquer alteração que forem fazer
- CTRL + c pra parar o servidor
